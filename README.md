### 天猫墙
[预览样式](https://donglei96.github.io/demo/天猫墙.html)
- 难点在于对于[0-22]的数组，如何拟定函数使
  - 序号0，返回0
  - 序号1，6返回1
  - 序号2，7，12返回3
  - 序号3，8，13，18返回3
  - 序号4，9，14，19返回4
  - ...
- 通过画坐标轴图样发现它们都是一次函数图样，得到函数 `y=x+(Math.floor(x/6))*-5`
